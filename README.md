# CodiceFiscale Ruby

[![Gem Version](https://badge.fury.io/rb/codicefiscale_ruby.svg)](https://badge.fury.io/rb/codicefiscale_ruby)

A Ruby gem for generating and validating Italian fiscal codes (Codice Fiscale).
